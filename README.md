# 3D Pasta Simulator

## Project Overview  
The 3D Pasta Simulator is an innovative tool that enables users to create and manipulate pasta shapes in a three-dimensional virtual environment. The simulator aims to provide a realistic experience that mimics the characteristics of actual pasta making, making it a useful software for both culinary professionals and enthusiasts.

## Features  
- **Realistic 3D Graphics:** High-quality rendering of various pasta shapes.
- **Interactive User Interface:** Simple and intuitive controls for creating pasta shapes.
- **Custom Shape Creation:** Option to design and modify pasta shapes as desired.
- **Simulation of Cooking Effects:** Visualize how different shapes interact with water and other ingredients.
- **Export Options:** Save designs for further use in cooking simulations or for educational purposes.

## Usage Guide  
1. **Installation:**  
   Download and install the software from the official repository.  
2. **Creating Pasta:**  
   Use the tools provided in the main menu to select and modify shapes.  
3. **Simulation:**  
   Run the simulation to see how the pasta behaves under various cooking conditions.  
4. **Exporting Designs:**  
   Save your creations for future reference or share with others.  

## Technical Details  
- **Programming Language:** Python
- **Frameworks Used:** Pygame for graphics, NumPy for mathematical operations.
- **Compiling:** Make sure to have Python 3.x installed.  
- **Dependencies:** Install required packages using `pip install -r requirements.txt` in your terminal.
